Module: server/services/pdf

PDF Service - PDF extraction and fetching utilities.

Handles PDF text extraction from buffers and fetching PDFs from Firebase Storage with automatic GitHub fallback for missing files.

(async, inner) extractTextFromPDF(pdfBuffer) → {Promise.<(string|null)>}

Extracts text content from a PDF buffer.

Performs basic text cleanup including:

  • Converting form feeds to newlines
  • Normalizing whitespace
  • Trimming excess blank lines

Raw PDF file buffer

(async, inner) fetchAndExtractPDFText(pdfFileName) → {Promise.<string>}

Fetches a PDF from Firebase Storage and extracts its text content.

If the PDF doesn't exist in Firebase Storage, automatically falls back to fetching from GitHub and uploads to Firebase for future use.
